# Binary Tree Lab (Лабораторная работа №4)

## Описание
Программа на Kotlin для работы с бинарным деревом поиска.

## Возможности
- Добавление / удаление узлов
- Поиск узла
- ASCII-вывод дерева
- Балансировка
- Статистика (высота, узлы, листья, min/max)

## Запуск
Открыть проект в IntelliJ IDEA → запустить `Main.kt`

## Структура
- `TreeNode.kt` – узел дерева
- `BinaryTree.kt` – логика дерева
- `Main.kt` – консольное меню
